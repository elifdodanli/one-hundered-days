## üìñ Task Explanations

Next.js provides several different ways to style your application. Today, you will explore the most common approaches. You will learn how to use global CSS for site-wide styles, CSS Modules for local, component-specific styles, and Styled-JSX for writing CSS directly inside your components. Understanding these options will allow you to choose the best styling strategy for your project.

## üí° What You'll Gain

The ability to style your Next.js applications using a variety of methods, from traditional CSS to modern CSS-in-JS solutions.

## üìö Dictionary for Developers

*   **Global CSS:** A CSS file that is applied to the entire application.
*   **CSS Modules:** A CSS file in which all class names and animation names are scoped locally by default.
*   **Styled-JSX:** A CSS-in-JS library that is built into Next.js, allowing you to write CSS within your React components.
*   **UI Library:** A collection of pre-built components that can be used to speed up development.
