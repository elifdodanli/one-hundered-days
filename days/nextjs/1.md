## ğŸ“š Dictionary for Developers

*   **React:** A JavaScript library for building user interfaces.
*   **JSX (JavaScript XML):** A syntax extension for JavaScript that lets you write HTML-like code in your JavaScript files.
*   **Component:** A reusable, self-contained piece of UI.
*   **Props (Properties):** A mechanism for passing data from a parent component to a child component.
*   **State:** An object that represents the parts of the app that can change. Each component can have its own state.
