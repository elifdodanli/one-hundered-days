## üìñ Task Explanations

Search Engine Optimization (SEO) is the practice of improving your site's visibility on search engines like Google. Next.js is an excellent framework for SEO because it makes it easy to create fast, server-rendered pages. Today, you will learn how to use the built-in features of Next.js to improve your site's SEO, including setting metadata, generating a sitemap, and creating a `robots.txt` file.

## üìö Dictionary for Developers

*   **SEO (Search Engine Optimization):** The process of improving the quality and quantity of website traffic to a website or a web page from search engines.
*   **Metadata:** Data that provides information about other data. In the context of a web page, this includes the title, description, and other information that appears in search engine results.
*   **Sitemap:** A file where you provide information about the pages, videos, and other files on your site, and the relationships between them.
