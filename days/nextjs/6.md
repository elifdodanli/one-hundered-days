API Routes allow you to easily create a backend for your Next.js application. Any file inside the `pages/api` directory is mapped to `/api/*` and will be treated as an API endpoint instead of a page. This allows you to build full-stack applications within a single Next.js project. Today, you will create your first API route and learn how to handle different types of requests.
