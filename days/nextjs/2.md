## üìñ Task Explanations

Hooks are a powerful feature that allows you to "hook into" React's state and lifecycle features from functional components. Today, you will learn about the most common Hooks. `useState` allows you to declare state variables, `useEffect` allows you to run code in response to component rendering or state changes, and `useContext` allows you to easily access global data.

## üí° What You'll Gain

The ability to write powerful, modern functional components in React using Hooks. This is a critical skill for any React developer.

## üìö Dictionary for Developers

*   **Hook:** A special function that lets you "hook into" React features. For example, `useState` is a Hook that lets you add React state to functional components.
*   **`useState`:** A Hook that lets you add a state variable to your component.
*   **`useEffect`:** A Hook that lets you perform side effects in your components.
*   **`useContext`:** A Hook that lets you read and subscribe to context from your component.
